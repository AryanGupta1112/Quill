# Quill

Quill is an Android notes application built with Kotlin.

## Features

- Create, edit, and delete notes
- Persistent local storage using Room
- RecyclerView-based notes list UI

## Tech Stack

- Kotlin
- Android SDK
- Room (SQLite)
- Gradle (KTS)

## Getting Started

1. Open the project in Android Studio.
2. Sync Gradle dependencies.
3. Run the `app` module on an emulator or device.

## Package

`com.example.quill`
